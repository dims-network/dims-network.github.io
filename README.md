# dims-network.github.io

Landing page for the **DIMS-network** — open tools for visualizing dynamic
interaction and multimodal signals in social-interaction research.

Live at **https://dims-network.github.io/**

The site (`index.html`, self-contained) shows the repositories in the network
and a canvas diagram of how they relate (forks and templates):

- **DIMS_Dashboard** — canonical static dashboard (base)
- **DIMS_dashboard_template** — empty "Use this template" starter
- **DIMS_Dashboard_Ortho** — fork of the base for the ORTHO study
- **bots-viewer** — Dash trajectory + RQA/cRQA viewer (base)
- **Ortho-viewer** — fork of bots-viewer (ORTHO Game Browser)

Reference: Miao, Trujillo, Bulls, Thornton, Dale, & Pouw (2025), *DIMS Dashboard
for Exploring Dynamic Interactions and Multimodal Signals*, CogSci 2025.
